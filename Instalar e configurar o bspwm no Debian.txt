3 - Os passos para configurar e usar o gpm no TTY

Instalar: apt install gpm

No TTY (modo texto puro) do Debian, o processo de copiar e colar funciona de maneira diferente dos ambientes gráficos. Como o TTY não tem suporte a interfaces gráficas comuns, você pode utilizar o gpm (General Purpose Mouse), que permite o uso do mouse para copiar e colar texto no console.

Iniciar o gpm: systemctl start gpm
Ativar o gpm: systemctl enable gpm

Copiar e colar:

੶ Para copiar texto: Selecione o texto com o mouse.
੶ Para colar o texto copiado: Clique com o botão direito do mouse no local onde deseja colar o texto.

O comando 'cat' simplesmente imprime o conteúdo de um arquivo ou a saída de um comando no terminal de uma só vez, sem permitir navegação interativa. Se o conteúdo for maior que a tela do TTY, ele rolará automaticamente para cima, sem a possibilidade de voltar ou descer a página.

Para visualizar o conteúdo exibido por cat e poder rolar para cima e para baixo, você pode usar em conjunto com 'less' para ter navegação controlada.

Exemplo: cat arquivo.txt | less

Ao usar o 'less', você pode utilizar as seguintes teclas de navegação.

੶ j: Desce uma linha
੶ k: Sobe uma linha
੶ u: Sobe meia página
੶ d: Desce meia página
੶ q: Sai da visualização e retorna ao terminal

10 - Configurando a Swap em zRAM

1. Crie ou edite o arquivo de configuração: sudo nano /etc/systemd/zram-generator.conf

Adicione o seguinte conteúdo:

[zram0]
zram-size = min (ram / 1, 2048)
compression-algorithm = zstd
swap-priority = 100


੶ zram-size: Define o tamanho da zRAM (swap)
੶ compression-algorithm: Algoritmo de compressão, o 'zstd' se destaca como a melhor opção
੶ swap-priority: Define a prioridade da zRAM no sistema de swap

Pressione CTRL + O para salvar o arquivo

Pressione CTRL + X para sair do nano

18 - Bluetooth

1. Instalar os pacotes necessários

sudo apt install bluetooth bluez blueman libspa-0.2-bluetooth

bluetooth e bluez: Gerenciam o funcionamento do Bluetooth no sistema.
blueman: Fornece uma interface gráfica para o Bluetooth.
libspa-0.2-bluetooth: Necessário para conexões de áudio Bluetooth, como fones de ouvido.

2. Ativar e habilitar o serviço Bluetooth

sudo systemctl enable bluetooth
sudo systemctl start bluetooth

3. Verifique se o serviço está ativo

systemctl status bluetooth

4. Configurar o ícone de Bluetooth na Polybar

Criar o módulo de Bluetooth na configuração da Polybar Abra o arquivo de configuração da Polybar localizado em ~/.config/polybar/config e adicione o seguinte módulo:

[module/bluetooth]
type = custom/script
exec = bluetoothctl show | grep -q "Powered: yes" && echo " On" || echo " Off"
interval = 5
click-left = bluetoothctl power on
click-right = bluetoothctl power off

exec: Verifica se o Bluetooth está ligado e exibe "On" ou "Off".
click-left e click-right: Ativam e desativam o Bluetooth clicando no ícone.

5. Adicionar o módulo na Polybar Na seção modules-right

modules-right = bluetooth ...

6. Salvar e reiniciar a Polybar Após salvar o arquivo, reinicie a Polybar

polybar-msg cmd restart

7. Configurar a confirmação de solicitações de emparelhamento ou recebimento de arquivos

Para exibir uma janela de confirmação ao emparelhar dispositivos ou receber arquivos, você usará o Blueman.

Adicione o seguinte comando no arquivo bspwmrc para iniciar o Blueman ao entrar no sistema: blueman-applet &

Esse comando inicializa o applet do Blueman, que gerencia conexões Bluetooth e exibe notificações interativas para emparelhamento e transferência de arquivos.

Salve as alterações no arquivo bspwmrc e reinicie o sistema ou apenas o BSPWM: bspc wm --restart

Ao emparelhar um dispositivo ou receber um arquivo, uma janela será exibida com a opção de aceitar ou rejeitar a solicitação.

8. Testar a configuração

Ligue o Bluetooth clicando no ícone da Polybar ou digitando no terminal: bluetoothctl power on

Certifique-se de que o applet do Blueman está ativo. Ele mostrará notificações interativas para emparelhamento e transferência de arquivos.

Firefox

1. Crie um diretório para armazenar chaves do repositório APT, se ainda não existir:

sudo install -d -m 0755 /etc/apt/keyrings

2. Importe a chave de assinatura do repositório APT da Mozilla:

wget -q https://packages.mozilla.org/apt/repo-signing-key.gpg -O- | sudo tee /etc/apt/keyrings/packages.mozilla.org.asc > /dev/null

3. O fingerprint deve ser 35BAA0B33E9EB396F59CA838C0BA5CE6DC6315A3. Você pode verificar com o seguinte comando:

gpg -n -q --import --import-options import-show /etc/apt/keyrings/packages.mozilla.org.asc | awk '/pub/{getline; gsub(/^ +| +$/,""); if($0 == "35BAA0B33E9EB396F59CA838C0BA5CE6DC6315A3") print "\nO fingerprint da chave corresponde ("$0").\n"; else print "\nFalha na verificação: o fingerprint ("$0") não corresponde ao esperado.\n"}'

4. Depois adicione o repositório APT da Mozilla à sua lista de origens:

echo "deb [signed-by=/etc/apt/keyrings/packages.mozilla.org.asc] https://packages.mozilla.org/apt mozilla main" | sudo tee -a /etc/apt/sources.list.d/mozilla.list > /dev/null

5. Configure o APT para dar prioridade a pacotes do repositório da Mozilla:

echo '
Package: *
Pin: origin packages.mozilla.org
Pin-Priority: 1000
' | sudo tee /etc/apt/preferences.d/mozilla

6. Atualize a lista de pacotes e instale o pacote .deb do Firefox:

sudo apt update && sudo apt install firefox-l10n-pt-br
