O Polybar é uma barra de status altamente personalizável, frequentemente usada com gerenciadores de janelas minimalistas como o bspwm. Para instalá-lo e configurá-lo no Debian, siga estas etapas:

1. Instalação do Polybar

Digite: sudo apt install polybar

2. Configurando o Polybar

1. Crie a pasta de configuração para o Polybar:

mkdir -p ~/.config/polybar

2. Copie o arquivo de configuração de exemplo (se disponível) ou crie o seu próprio:

cp /usr/share/doc/polybar/examples/config.ini ~/.config/polybar/config

Ou crie um novo arquivo ~/.config/polybar/config com o seguinte exemplo básico:

[bar/example]
width = 100%
height = 30
radius = 5.0
fixed-center = true

background = #222
foreground = #dfdfdf

modules-left = bspwm
modules-center = date
modules-right = pulseaudio

[module/bspwm]
type = internal/bspwm
label-focused = %name%
label-occupied = %name%
label-urgent = %name%

[module/date]
type = internal/date
interval = 60
date = %Y-%m-%d %H:%M:%S
label = %date%

[module/pulseaudio]
type = internal/pulseaudio
label-volume = %volume: 3%%

3. Integrando o Polybar ao BSPWM

Para iniciar o Polybar automaticamente ao iniciar o bspwm, edite o arquivo ~/.config/bspwm/bspwmrc e adicione a seguinte linha ao final do arquivo:

~/.config/polybar/launch.sh &

Agora, crie o script de inicialização launch.sh para iniciar a barra:

nano ~/.config/polybar/launch.sh

E adicione o seguinte conteúdo:

#!/bin/bash

# Mata instâncias anteriores do polybar
killall -q polybar

# Aguarda até que as instâncias sejam encerradas
while pgrep -u $UID -x polybar >/dev/null; do sleep 1; done

# Inicia o polybar
polybar example &

Dê permissão de execução ao script:

chmod +x ~/.config/polybar/launch.sh

4. Personalizando o Polybar

O arquivo de configuração do Polybar é altamente personalizável. Você pode adicionar módulos como monitoramento de rede, bateria, temperatura, notificações, etc. Cada módulo pode ser configurado com cores, ícones e formatos diferentes.

5. Executando o Polybar

Agora, ao iniciar o bspwm, o Polybar deve ser iniciado automaticamente. Se preferir, você também pode iniciar o Polybar manualmente com o comando:

~/.config/polybar/launch.sh

Com isso, o Polybar estará funcionando em conjunto com o bspwm, e você pode ajustá-lo conforme suas necessidades e preferências.
