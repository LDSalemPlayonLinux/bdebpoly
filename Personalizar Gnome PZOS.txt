Personalizar Gnome no Debian

1 - Instalar temas e ícones

sudo apt install gnome-themes-extra papirus-icon-theme

Tema MacOS (Claro): git clone https://github.com/B00merang-Project/macOS.git
Tema MacOS (Escuro): git clone https://github.com/B00merang-Project/macOS-Dark.git

Icone OS Catalina: https://www.gnome-look.org/p/1309810
Icone macOS Ventura: https://www.gnome-look.org/p/2023325

MacOS Pixel Cursor: https://www.gnome-look.org/p/2042592

Depois de baixar os arquivos, extraia-os e mova para as pastas corretas:

Icone

sudo mv Os-Catalina-icons /usr/share/icons/

sudo mv Os-Catalina-Night /usr/share/icons/

sudo mv BigSur-Originals-Colors-blue /usr/share/icons/

sudo mv BigSur-Originals-Colors-blue-dark /usr/share/icons/

Cursor

sudo mv MacOS-Pixel-vr5 /usr/share/icons/

Tema

sudo mv macOS-Dark /usr/share/themes/

sudo mv macOS /usr/share/themes/

Agora, você pode ativar os temas usando o GNOME Tweaks (Ajustes do GNOME).

Caso o GNOME Tweaks não esteja instalado, digite: sudo apt install gnome-tweaks

2 -  Instalar e ativar a extensão "Dash to Panel"

O Zorin OS usa um painel inferior parecido com o Windows. Para isso, instale o Dash to Panel: 

sudo apt install gnome-shell-extension-dash-to-panel

Depois, ative a extensão no GNOME Tweaks e configure para ficar semelhante ao Zorin.

3 - Instalar fonte

sudo apt install fonts-roboto

Depois, vá ao GNOME Tweaks > Aparência > Fonte e altere para Roboto Regular.

4 - Alterar o papel de parede

Baixe o wallpaper macOS Catalina

https://www.gnome-look.org/p/1362975/

5 - Configurar o layout do GNOME

No GNOME Tweaks, faça estas alterações:

1. Aparência

Tema: MacOS-Dark
Ícones: Papirus-Light
Cursor: MacOS-Pixel-vr5
Shell: MacOS

"Para alterar o Shell, é necessário ativar a opção 'User Themes'. Em outras distros com GNOME, é preciso instalar o pacote 'gnome-shell-extension-prefs'. No Debian, o aplicativo 'Extensões' já vem instalado, permitindo ativar a opção 'User Themes' diretamente.

2. Extensões

Ative o "Dash to Panel"

3. Teclas e Mouse

Ative "Tap to Click" no touchpad, se desejar

6 - Instalar o Arc Menu

sudo apt install gnome-shell-extension-arc-menu

Depois, ative pelo GNOME Tweaks em Extensões.

Configurar o Arc Menu 

1. Clique com o botão direito no ícone do menu e vá em Configurações.
2. Em Estilo do Layout, escolha Zorin OS.

Agora seu menu estará bem próximo do original do Zorin!
